Coding

In Spaceflight Simulator, I've created various scripts to manipulate game data saved in JSON-formatted text files.

One script procedurally generates a solar system by assigning ecosystems, textures, and moons to planets based on their distance from the sun.

Another script uses a simple AI to recreate user-inputted images for the game, generating an output image and a file for in-game loading. For instance, the middle picture of the Saturn V showcases this image recreation.

The last script is of a procedural space station maker, connecting structural parts through docking ports. Users can define the number of modules, and the script rotates and connects them accordingly.
